# DejaVu Sans (bundled fonts)

Файлы `DejaVuSans.ttf` и `DejaVuSans-Bold.ttf` используются:
- ReportLab для генерации PDF с кириллицей (FR-RPT-02);
- matplotlib для подписей на чартах sentiment / topics.

**Решение Q3 пользователя для Phase 4:** bundle TTF в репо вместо apt-install,
чтобы воспроизводимость отчётов не зависела от ОС-уровня и пакет-менеджера.
